# Server perfomance stats

This is a Bash script that analyzes basic server statistics such as CPU usage, RAM usage, load average, and disk usage. It also lists the top 5 CPU- and RAM-consuming processes.

## Installation Guide

### STEP 1
**Clone repository**
```
git clone https://github.com/xRezek/server_performance_stats.git
cd server_performance_stats
```
### STEP 2
**Add proper permissions**
``` 
chmod +x server-stats.sh
```
### STEP 3
**Run the script**
```
./server-stats.sh
```
The idea of this project is taken from [roadmap.sh](https://roadmap.sh/projects/server-stats)
